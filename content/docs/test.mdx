## Installation

This template uses **npm** as package manager. To get started:

```bash
npm install
npm run dev
```

Your site will be available at `http://localhost:3000`.

## Project Structure

```
.
├── app/                    # Next.js App Router
│   ├── (home)/            # Homepage
│   ├── docs/              # Documentation pages
│   └── layout.tsx         # Root layout
├── content/
│   └── docs/              # MDX documentation files
├── lib/
│   ├── source.ts          # Fumadocs source config
│   └── layout.shared.tsx  # Navigation config
└── source.config.ts       # MDX configuration
```

## Writing Documentation

Create `.mdx` files in `/content/docs`:

```mdx
---
title: My Page
description: Page description
---

## Heading

Your content here with **markdown** support.

<Cards>
  <Card title="Link" href="/docs/other-page" />
</Cards>
```

## Components

Fumadocs provides built-in components:

### Cards

```mdx
<Cards>
  <Card title="Card Title" href="/link" />
  <Card title="External Link" href="https://example.com" />
</Cards>
```

### Code Blocks

````mdx
```typescript
function hello(name: string) {
  return `Hello, ${name}!`;
}
```
````

### Callouts

```mdx
> [!NOTE]
> This is a note callout

> [!WARNING]
> This is a warning callout
```

## Customization

### Colors

Edit `app/global.css` to change the accent color:

```css
:root {
  --primary: 237 70% 67%; /* HSL format */
}
```

### Navigation

Edit `lib/layout.shared.tsx`:

```tsx
export function baseOptions(): BaseLayoutProps {
  return {
    nav: {
      title: 'Your Brand',
    },
    links: [
      { text: 'Docs', url: '/docs' },
      { text: 'GitHub', url: 'https://github.com/...', external: true },
    ],
  };
}
```
