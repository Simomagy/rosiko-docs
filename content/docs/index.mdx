Welcome to the **Rosiko** documentation! This site contains comprehensive technical documentation for a strategic multiplayer Risk-like game developed in Unreal Engine 5.

## About Rosiko

**Rosiko** is a turn-based strategy multiplayer game inspired by Risk, featuring:

- **Objectives System** - Main and secondary objectives with secret assignment
- **Server-Authoritative Multiplayer** - Secure client-server architecture
- **Procedural Map Generation** - Deterministic territory distribution
- **Enhanced Input System** - Modern UE5 input handling
- **Real-time Replication** - Optimized networking with conditional replication
