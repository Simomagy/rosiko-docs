## ğŸ† Highlights Tecnici

**Server-Authoritative Architecture**
Tutte le decisioni di gameplay sul server. Client inviano richieste, server valida.

**Conditional Replication (COND_OwnerOnly)**
Obiettivi segreti replicati solo al proprietario. Sicurezza garantita a livello engine.

**RepNotify + Event Dispatcher**
UI si aggiorna automaticamente quando dati cambiano. Zero polling, massima reattivitÃ .

**Data-Driven Design**
Obiettivi, continenti, territori configurabili tramite DataAssets. Nessun hardcoding.

**Deterministic Generation**
Mappa e obiettivi generati con seed fisso. Stessi risultati su tutti i client.

**Type-Safe C++**
